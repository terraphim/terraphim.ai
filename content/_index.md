Terraphim is a local-first knowledge graph search engine built in Rust and WebAssembly.
Your data never leaves your machine. Aho-Corasick pattern matching across role-based
knowledge graphs, at native speed.

Terraphim provides sub-millisecond search latency using Aho-Corasick multi-pattern matching.
It runs on your hardware as a desktop application, command-line tool, HTTP server, or browser extension.
No cloud dependency, no telemetry, no account required.

Core capabilities include local-first architecture, knowledge graph engine, MCP server integration,
Rust and WASM performance, hooks system for AI coding assistants, and KG-boosted file search.
Terraphim supports role-based knowledge graphs for system operators, engineers, and analysts.

Install via Homebrew, Cargo, curl quick install, or download binaries for macOS, Linux, and Windows.
Library bindings available for Node.js via @terraphim/autocomplete, Python via terraphim-automata,
and a browser extension with WASM-powered search.
